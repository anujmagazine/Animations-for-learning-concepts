# AI&Beyond — Interactive Business Model Canvas

An interactive, click-to-explore Business Model Canvas for [AI&Beyond](https://aiandbeyond.ai), India's leading enterprise AI literacy company. Built as a single-file HTML artifact — no dependencies, no build tools, no frameworks.

🔗 **[View Live →](https://YOUR_GITHUB_USERNAME.github.io/aiandbeyond-bmc/)**

---

### What is this?

A visual breakdown of AI&Beyond's business model across all nine dimensions of the Business Model Canvas:

| Dimension | What it covers |
|---|---|
| **Key Partners** | Ashoka University, IndiaAI Mission, Inflexion, Coursera |
| **Key Activities** | Custom bootcamp design, keynotes, LaaS, consulting |
| **Key Resources** | Founder brand, proprietary frameworks, scenario IP |
| **Value Propositions** | AI literacy as 4th pillar, workflow-first, industry-customised |
| **Customer Relationships** | High-touch founder-led, pre-bootcamp questionnaires, LaaS retainers |
| **Channels** | Direct sales, LinkedIn, newsletter, YUVA AI, book |
| **Customer Segments** | Enterprise C-suite, function heads, govt & defence, India + MEA + SEA |
| **Cost Structure** | Founder time, AI tools, content development, ops team |
| **Revenue Streams** | Bootcamp fees, LaaS retainers, keynotes, consulting, book royalties |

### Features

- **Click any block** to expand detailed insights with colour-coded tags
- **Light & dark mode** — auto-adapts to your system preference
- **Zero dependencies** — single HTML file, works offline
- **Responsive grid** — standard Business Model Canvas layout with 10-column CSS grid
- **Mobile-friendly** — tap to explore on any device

### Quick Start

#### Option 1: GitHub Pages

1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your canvas is live at `https://YOUR_USERNAME.github.io/aiandbeyond-bmc/`

#### Option 2: Open locally
```bash
git clone https://github.com/YOUR_USERNAME/aiandbeyond-bmc.git
open index.html
```

No server needed. Just open the HTML file in any browser.

### About AI&Beyond

[AI&Beyond](https://aiandbeyond.ai) is an enterprise AI literacy company co-founded by **Jaspreet Bindra**, **Anuj Magazine**, and **Archana Muthappa**. The company helps organisations build AI literacy across every level — from boardroom strategic briefings to frontline workflow transformation.

**Core thesis:** AI literacy is the fourth pillar of modern literacy, alongside reading, writing, and arithmetic.

**What makes it different:**
- Bootcamps built around YOUR workflows, YOUR industry, YOUR challenges — not generic demos
- 45+ enterprise clients across aviation, financial services, legal, real estate, chemicals, sports, logistics, insurance, and professional services
- Co-created the Government of India's **YUVA AI for All** course under the IndiaAI Mission, targeting 10 million citizens
- Authors of **"Winning with AI: Your Guide to AI Literacy"** — the first AI book with its own AI chatbot

**Clients include:** IndiGo Airlines, KPMG, JSA Law, Sun Life, Capri Global, Crown Worldwide, Ajmera Realty, OGQ, ASCI, DCW, Shriram Life Insurance, and others.

**Geography:** India (core) + Middle East + Southeast Asia

### How It Was Built

This canvas was generated using **Claude** (Anthropic) as an interactive visualisation artifact, then exported as a standalone HTML file. The data was compiled from:

- AI&Beyond's official website and public communications
- Press coverage (ANI News, CXOToday, India Today, APN News)
- Government of India press releases (PIB) on the YUVA AI initiative
- BusinessABC company profile
